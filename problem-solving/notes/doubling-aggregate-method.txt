If n (the number we stop at) is 2^x + 1
1 + 2 + 4 + 8 + .... + n - 1
= 2^0 + 2^1 + 2^2 + 2^3 + ... + 2^log(n - 1)

n

2 * 2^log(n - 1) - 1 + n
= 3n - 3

n
